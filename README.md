**QR Scanner and Generator App**



A high-performance Android QR Scanner and Generator application built with Kotlin. This app is designed with a native UI inspired by iOS, providing a clean, minimal, and offline-first experience.



**Platform and Tech Stack**

* Language: Kotlin
* Platform: Android
* Backend: None (Completely Offline)
* Authentication: None
* UI Framework: Custom native UI (Material Design components disabled)
* Architecture: Lightweight and performance-focused



**Design and UI Guidelines**

* UI Style: iOS-inspired native look
* Color Scheme: Sky Blue, White, and Black
* Material Design: Disabled completely
* Animations: Super smooth transitions with a target of 120 FPS
* UX Focus: Clean, minimal, and free of visual clutter



**App Structure**

* Splash Screen

&nbsp;Minimal logo screen

&nbsp;Smooth fade-in and fade-out animations

&nbsp;Quick transition to the main application



* Bottom Navigation

&nbsp;Fixed bottom navigation bar with iOS-style icons

&nbsp;Smooth tab switching animations

&nbsp;Banner advertisement placed strategically above the navigation bar



**Home Screen**

* QR Code Generator

&nbsp;Generate QR codes for text, URLs, and other supported content types.



* QR Code Scanner

&nbsp;Fast and accurate camera scanning

&nbsp;Stable performance across various lighting conditions



* After Scan Result

&nbsp;Clear display of scanned content

&nbsp;Context-based actions such as opening URLs or redirecting to websites

&nbsp;Options to copy and share content



**History Screen**

* Stores all past QR activity locally on the device.
* Stored Data
* Scanned and generated QR codes
* Content details with date and time stamps
* Categorization by type (Scanned or Generated)



**Features**

* Chronological history list
* Filter options located below the App Bar including Scanned, Generated, Date, and Content Type
* Analytics and Visualization
* Graphs showing the number of scans and generated QR codes
* Visualization of usage trends
* Clear and readable analytics interface
* 100% local storage with no cloud synchronization



**Settings Screen**

Dark Mode and Light Mode toggle

Terms and Conditions

About Application section featuring the app name, version, and developer information





**How to Run**

* Clone the repository.
* Open the project in Android Studio.
* Sync Gradle files.
* Run the application on an emulator or a physical device.
